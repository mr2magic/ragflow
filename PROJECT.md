## Standards

**Quality:**
- ! Run `task check` before every commit
- ! Achieve ≥85% coverage overall + per-module
- ! Store secrets in `secrets/` dir or `docker/.env` (never commit)
- ~ Provide `.example` templates for secrets

## Project-Specific Rules

- ! `docker/.env` is untracked — never commit it (contains API keys and passwords)
- ~ New agent tools go in `agent/tools/` following the `ToolParamBase` / `ToolBase` pattern
- ~ Docker changes should be tested against `docker-compose-macos.yml` locally
