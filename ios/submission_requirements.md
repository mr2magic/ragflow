# App Store Submission Requirements

## Description (App Store → App Information → English)

Ragion turns your documents into a searchable knowledge base you can chat with using AI.

Import PDFs, ePubs, Word, Excel, PowerPoint, emails, Markdown, code files, and more — from Files, iCloud, or a URL. The app chunks and embeds your documents locally, then lets you ask questions and get cited answers.

FEATURES
• Chat with your documents using Claude, OpenAI, or local Ollama models
• Import 20+ file formats including PDF, ePub, DOCX, XLSX, PPTX, EML, HTML, CSV, JSON, YAML, Swift, Python, JS, Go, SQL, and shell scripts
• Multiple knowledge bases — organize documents by project or topic
• Build multi-step AI workflows with retrieval, LLM, and summarization steps
• Full-text search across all indexed passages
• Share or export chat history
• Runs entirely on-device with Ollama — no data leaves your iPhone

## Notes

### Privacy Policy
The app does not collect, transmit, or share any user data. All document processing happens on-device.
A minimal privacy policy hosted at a public URL is required. Options:
- GitHub Gist (free, permanent URL)
- A `/privacy` page in the repo's GitHub Pages site
- Any public URL you control
